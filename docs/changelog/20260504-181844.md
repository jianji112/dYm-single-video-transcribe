# 2026-05-04 18:18 — 图集自动切换

## 任务描述
图集类型作品在播放/查看时每 3 秒自动切换到下一张；用户一旦手动操作（左右箭头、键盘 ←→、点击指示点），当前作品本次播放停止自动切换；切换到其他作品或重新打开浏览器/Viewer 时重置。Web feed 与 Electron 客户端 MediaViewer 同步加上。

## 行为
| 触发 | 动作 |
|---|---|
| 切到图集作品 | 3s 自动切下一张，循环 |
| 手动按左右 / 点指示点 / 键盘 ← → | 当前作品本次停止自动切换 |
| 切到下一作品再切回 | 仍保持已停止（同次播放） |
| 关闭 player / Viewer 重新打开 | 重置 |
| 视频作品 | 不受影响 |
