# 屏蔽 Windows 平台 bytedance 协议弹窗

## 问题
Windows 平台上打开抖音登录窗口或访问用户主页时，抖音网页尝试导航到 `bytedance://` 自定义协议，触发 Windows 系统弹窗"获取打开此 bytedance 链接的应用"。

## 方案
在所有加载抖音页面的 BrowserWindow 上拦截 `bytedance:`、`snssdk:`、`aweme:` 等自定义协议的导航，阻止系统弹窗。
